Lodestar — 核心概念動圖

三支動畫：① 整條 pipeline 的產出怎麼流動（給所有人看）② workflow / stage 的架構精神（可操作模擬）③ 工廠分析 RCA Copilot（領域 plugin 示範）。支援深色模式，瀏覽器直接開即可。

① 從一句話到合併的 PR — 產出流動總覽

綠色卡片都是「產出」。自動播放，也可用按鈕逐步看。

② workflow / stage 的架構精神 — 可操作模擬器

點卡片＝生成；已核准再點＝重生成（下游自動 reset）；點上鎖的卡看 engine 擋下。

1 · plugin 註冊能力卡（StageSpec 掛進 registry，裝／卸像 App）

prd
depends_on = ()
generate · refine · chat
architecture
depends_on = ("prd",)
generate · refine · chat
stories
depends_on = ("architecture",)
generate · refine · chat
change_request
depends_on = ()
requires = ("workspace",)

2 · workflow 是一筆資料（WorkflowSpec 存 DB；換流程＝換資料，engine 不變）

WorkflowSpec(id="default", stages=("prd", "architecture", "stories"))

提示順序：先點上鎖的「架構」看被擋下 → 依序生成到全綠 → 回頭重生成 PRD，看下游被 reset。

3 · engine console — 零 stage 硬編碼，依賴與下游 reset 全從資料推導

③ 工廠分析 — RCA Copilot（領域 plugin）
